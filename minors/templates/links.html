Systems Links (Production)

Minor/Certificate Credential Request Form

  • - Allows colleges to request new minors/certificates.

Minors/Certificate Add/Update/View/Search

  • - Allows anyone with a UT EID to search for all minors/certificates.
  • - Allows registrar staff to add and update minor/certificate info.

Degree Plans Add/Update/View/Search

  • - Allows authorized staff to search and view existing degree plans, minors, and certificates.
  • - Allows registrar staff to add and update degree plans, minors, and certificates.

Minor/Certificate Applications and Approvals

  • - Allows students to apply for minors/certificates.
  • - Allows advisors to apply for minors/certificates for a student.
  • - Allows colleges/departments the ability to approve students in restricted minors/certificates.

Student IDA Profiles

  • - Allows advisors to link minors/certificates to a student’s major degree plan.

IDA 2.0

  • - Allows anyone with a UT EID to run degree audits on degree plans, minors, and certificates.
  • - Allows colleges to certify degree plans with minors/certificates.

Linking your page directly to the student application

You may link directly into the specific minor or certificate page from your website by using the formats below. Note the changes after the 'submission/'section.
Examples:

  • Open Minors: https://utdirect.utexas.edu/apps/sisutil/edoc/submission/applyforminor?action=2&minors=MNCADV01
  • Application for a Minor: https://utdirect.utexas.edu/apps/sisutil/edoc/submission/regforminor?action=2&minors=MN9ARI01
  • Open Certificate: https://utdirect.utexas.edu/apps/sisutil/edoc/submission/applyforcertificate?action=2&minors=CTEBIO01
  • Application for a Certificate: https://utdirect.utexas.edu/apps/sisutil/edoc/submission/regforcertificate?action=2&minors=CTUBDP01
